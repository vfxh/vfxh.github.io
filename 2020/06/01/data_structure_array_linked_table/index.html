数组

数组是一种数据类型相同，长度固定，顺序存储的数据结构。也就说初始化的时候，会分配一块固定大小的内存，元素的存储时连续的。

优点

  1. 在下标范围内支持快速随机读取和更新数据，时间复杂度为O(1)

缺点

  1. 在插入和删除元素的时候需要移动元素位置。时间复杂度为O(n)
  2. 长度固定，想要存储更多元素的时候需要扩容

数组结构体声明
因为需要扩容，所以不直接使用数组，而是封装成一个结构体

插入数据

  1. 向数组增加元素的时候，需要先判断插入的位置是否合法。插入的位置可以是第一个位置（下标为0的位置），也可以是中间某个位置，或者最后一个位置，还可以是最后一个元素的下一个位置。但是不能插入除了这些位置之外的地方，因为数组是连续的，新插入的元素类原来的元素必须连续。
  2. 在除了插入最后一个元素位置的下一个位置的情况下，数组元素都需要移动，给新的元素留出位置
  3. 如果当前数组已经没有空间了，那么就重新分配一块更大的内存。然后将旧内存的数据拷贝到新内存上，之后释放旧内存。
